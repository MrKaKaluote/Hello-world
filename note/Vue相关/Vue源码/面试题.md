## vue2和vue3的区别

1. vue3使用ts编写，对ts的支持更友好
2. vue3使用proxy对数据进行响应式监听，性能要比vue2高很多
3. vue3使用compomentsApi进行开发，提高了开发的灵活性
4. vue3对diff算法进行了优化
